An explicit list of features

  • Single Agent Navigation: The agent moves at a constant velocity along its path from the start to goal position. I've also implemented motion smoothing as shown in the lecture by skipping nodes in the path if possible
  • Improved Agent and Scene Rendering: The theme is a plane navigating throughout the clouds. the clouds are sprites that are used as the texture of a quad. an airplane sprite is also used. Bounding circles are used as the hitbox
  • Orientation Smoothing: the planes orientation matches its path. The motion is also smoothed.
  • User Scenario Editing - obstacles can be added to the Scene at the mouse positionby pressing the A key on the keyboard. an obstacle can be moved around by hovering over it with the mouse and using the arrow keys. the agent position can be moved with right click and the goal position can be moved with left click
